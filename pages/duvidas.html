Dúvidas Frequentes (FAQ)

Voltar para a Página Principal

Como faço para encontrar um aluno?

Use a barra de busca no topo da página digitando o nome, número ou curso do aluno.

Consigo usar o sistema no celular?

Sim! O Carógrafo é responsivo e funciona em dispositivos móveis e computadores.

Os dados dos alunos ficam armazenados?

Não. O sistema apenas exibe os dados públicos e não coleta nenhuma informação pessoal dos usuários.

Como os alunos são cadastrados?

Os alunos são adicionados por meio de um arquivo JSON/planilha/API interna. A exibição dos cards é automática.

Quem desenvolveu o sistema?

O Carógrafo foi desenvolvido pelos alunos do curso técnico do CEFET-MG. Veja mais na página "Alunos Desenvolvedores".
