O que é o Carógrafo

O Carógrafo é um sistema desenvolvido para facilitar a visualização e a organização de informações sobre os alunos do CEFET-MG. A plataforma permite acessar rapidamente dados como nome, turma, curso e foto dos estudantes, com uma interface simples, intuitiva e funcional.

Objetivo do Projeto

O objetivo principal do Carógrafo é substituir os antigos quadros físicos e arquivos estáticos por uma solução digital mais eficiente, acessível e moderna. Com isso, buscamos oferecer uma ferramenta útil tanto para alunos quanto para professores e coordenação.

Funcionalidades principais

  • Listagem de alunos com foto e dados básicos
  • Sistema de busca por nome
  • Paginação para facilitar a navegação
  • Interface responsiva (adaptada para celular e computador)
  • Integração com dados dinâmicos (futuramente via API)

Desenvolvimento

O projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) do curso técnico de Informática para Internet do CEFET-MG. Utilizamos tecnologias como HTML, CSS, Tailwind CSS e JavaScript para a construção da interface e lógica do sistema.
